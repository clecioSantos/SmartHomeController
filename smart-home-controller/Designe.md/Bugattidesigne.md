# Bugatti Dashboard Design System

Este documento define a estética de hiper-luxo e performance para o SmartHome Controller.

## 1. Paleta de Cores (Couture)

- **Primary (Bugatti Blue):** `#004b93`
- **Accent (Ettore Red):** `#e10600`
- **Background (Carbon Black):** `#050505`
- **Surface (Deep Gray):** `#111111`
- **Border (Machined Silver):** `#222222`
- **Text Primary:** `#ffffff`
- **Text Secondary:** `#888888`

## 2. Estética

- **Bordas:** Sharp ou levemente arredondadas (4px a 8px) evocando precisão técnica.
- **Sombras:** "Inner glows" sutis e sombras profundas para criar volumetria.
- **Tipografia:** Negritos fortes e tracking condensado para títulos.

## 3. Componentes

### Cards de Dispositivo
- Fundo: Gradiente sutil de `#111111` para `#0a0a0a`.
- Destaque: Linha lateral ou brilho na cor Bugatti Blue quando ativo.

### Controles
- Toggles e botões com acabamento metálico ou cores sólidas vibrantes sobre fundo escuro.

## 4. Layout

- Dashboard imersivo, ocupando a tela com alto contraste.
- Grid rígido e organizado.
