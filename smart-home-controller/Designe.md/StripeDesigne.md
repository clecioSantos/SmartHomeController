# Stripe Dashboard Design System

Este documento descreve as diretrizes de design extraídas para a integração do estilo Stripe no SmartHome Controller.

## 1. Cores (Palette)

- **Primary:** `#635bff` (Stripe Blurple)
- **Background:** `#f6f9fc` (Light Grey/Blue)
- **Dark Background:** `#0a2540`
- **Text Primary:** `#1a1f36`
- **Text Secondary:** `#4f566b`
- **Success:** `#00d924`

## 2. Tipografia

- **Font Family:** 'Inter', -apple-system, sans-serif.
- **Headings:** Semibold, tracking -0.01em.
- **Body:** Regular, line-height 1.6.

## 3. Componentes Principais

### Sidebar
- Background: White.
- Icons: Estilo linha (Lucide), cinza neutro, estado ativo em Blurple.
- Width: 280px.

### Dashboard Cards
- Shadow: `0 7px 14px 0 rgba(60,66,87, 0.08), 0 3px 6px 0 rgba(0,0,0, 0.12)`.
- Border Radius: `8px`.
- Border: `1px solid #e3e8ee`.

### Buttons
- Primary: Background Blurple, Text White, `box-shadow` suave no hover.
- Ghost: Fundo transparente, texto Blurple, borda visível apenas no hover.

## 4. Layout

- Grid de 12 colunas.
- Espaçamento (Gutter): 24px.
- Margens laterais: 40px em telas desktop.
